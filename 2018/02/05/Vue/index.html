Vue的生命周期

钩子函数 描述
beforeCreate 实例刚被创建，数据、属性、事件的计算绑定之前
created 实例创建完成，数据、属性、事件已计算绑定，但DOM还未生成，$el属性还不存在
beforeMount 模板编译/挂载之前
mounted 模板编译/挂载之后。不保证所有子组件均挂载
beforeUpdate 组件更新之前
updated 组件更新之后
activated for keep-alive，组件被激活时调用
deactivated for keep-alive，组件被移除时调用
beforeDestroy 组件销毁前调用。此时的实例仍然完全可用
destroyed 组件销毁后调用
